# DSN-POP32 Library

A utility library for **POP32** robots developed for the **DSN Robot** club, designed to simplify development and improve ease of use.

Primarily intended for:
- Line-following robots
- Mission robots
- Item-dropping robots

## Hardware Compatibility

### IMU Sensor
- ZX-IMU (INEX)

### Line Tracking Sensor
- Any analog sensor with single value per port

### Motor
- 2-motor drive control
- 4-motor drive control
